# Energy Forecasting Project (ARIMA)

## Overview
This repository contains a Jupyter notebook **`Energy_Forecasting_Project_ARIMA.ipynb`** that demonstrates how to use the ARIMA (AutoRegressive Integrated Moving Average) model for forecasting energy consumption.

## Features
- Load and preprocess energy consumption data.
- Perform exploratory data analysis (EDA) with visualizations.
- Build and evaluate an ARIMA model.
- Forecast future energy usage and visualize predictions.

## Requirements
- Python 3.9+ 
- Jupyter Notebook
- pandas
- numpy
- matplotlib
- statsmodels
- seaborn (optional for additional plots)

You can install the required packages via:
```bash
pip install pandas numpy matplotlib statsmodels seaborn jupyter
```

## Usage
1. Clone the repository:
```bash
git clone <repository-url>
cd Energy_Forcasting_Project
```
2. Launch Jupyter Notebook:
```bash
jupyter notebook
```
3. Open `Energy_Forecasting_Project_ARIMA.ipynb` and run the cells sequentially.
